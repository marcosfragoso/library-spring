# Biblioteca 
## Descrição
Projeto de um sistema de uma biblioteca simples, na qual são cadastrados autores, livros, categorias, editoras, usuários e os registros dos seus empréstimos.
## Ferramentas utilizadas
* Java
* Spring Boot
* JPA
* Hibernate
* H2 Console
* Maven
